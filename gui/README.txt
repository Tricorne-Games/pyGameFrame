This folder has nothing to do with the game, but is instead a supplementary boilerplate (like the rest of pyGameFrame), to create a wxPython-based GUI application, typically something like an editor or some other tool.

main.py - The main script.

app.py - The application file.

appframe.py - The file that describes the wx.Frame object used to run the app.

contextmenu.py - A sample context menu for right-clicking.

mainmenu.py - A quick setup for the top main menu.

testpanel.py - A quick setup for the UI layout. Defines a wx.Panel.
